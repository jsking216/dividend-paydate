# Dividend Paydate

Dividend Paydate is a Node app for trading equities based on the Dividend Paydate Effect.

## Installation

```bash
npm install
npm install --save-dev @babel/core @babel/cli @babel/preset-env @babel/node nodemon
```

## Usage

```bash
npm start
```
